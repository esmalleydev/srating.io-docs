## Connect to our API in a 3 steps
Use our API to get [live scores](/game/getGames), [live odds](/game/getOdds), [statistics](/team/getStats), [teams](/team/get), [ranking](/ranking/get) and more!


### 1. Get a free trial or paid API key
[View our plans here...](https://srating.io/pricing) all you need is an email address. The free trial is limited to 500 requests per month and some data may be scrambled.

### 2. Required Headers
```console
X-API-KEY
```
You must include your API key in the header under "X-API-KEY"

### 3. Specify the sport and division

Most requests will require a sport (organization_id) and division. For example games for college basketball and football are stored in the same table. [Read more here](/sports)

### 4. Send a request

[Using python? Click here](/python)

GET

```console
https://api.srating.io/v1/game/getGames?start_date=2023-04-03&organization_id=f1c37c98-3b4c-11ef-94bc-2a93761010b8&division_id=bf602dc4-3b4a-11ef-94bc-2a93761010b8
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'game',
    'function': 'getGames',
    'arguments': {
      'organization_id': 'f1c37c98-3b4c-11ef-94bc-2a93761010b8',
      'division_id': 'bf602dc4-3b4a-11ef-94bc-2a93761010b8',
      'start_date': '2023-04-03',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
