## Get a ranking row

This request will find the first matching row and typically only used when passing in the `cbb_ranking_id` or `team_id` and `season` argument.

### Request

GET

```console
https://api.srating.io/v1/cbb_ranking/get?team_id=87019264-8549-11ed-bf01-5296e1552828&season=2023&current=1
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'cbb_ranking',
    'function': 'get',
    'arguments': {
      'team_id': '87019264-8549-11ed-bf01-5296e1552828',
      'season': 2023,
      'current': 1,
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
