## Get a boxscore row

This request will find the first matching row and typically only used when passing in the `cbb_boxscore_id` or `cbb_game_id` and `team_id` argument.

### Request

GET

```console
https://api.srating.io/v1/cbb_boxscore/get?cbb_game_id=b58918bc-d155-11ed-9185-b6be2f39279c&team_id=75a9cb12-8549-11ed-bf01-5296e1552828
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'cbb_boxscore',
    'function': 'get',
    'arguments': {
      'cbb_game_id': 'b58918bc-d155-11ed-9185-b6be2f39279c',
      'team_id': '75a9cb12-8549-11ed-bf01-5296e1552828',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
