## Get a play by play row

This request will find the first matching row and typically only used when passing in the `cbb_game_pbp_id` argument.

### Request

GET

```console
https://api.srating.io/v1/cbb_game_pbp/get?cbb_game_pbp_id=ff2b0489-bee5-11ed-9185-b6be2f39279c
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'cbb_game_pbp',
    'function': 'get',
    'arguments': {
      'cbb_game_pbp_id': 'ff2b0489-bee5-11ed-9185-b6be2f39279c',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
