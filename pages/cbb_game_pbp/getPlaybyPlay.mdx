## Get all the play by play rows for a game

This request will grab all the play by play rows. Other filters can be used in addition to the cbb_game_id

### Required arguments
`cbb_game_id`

### Request

GET

```console
https://api.srating.io/v1/cbb_game_pbp/getPlaybyPlay?cbb_game_id=d85cdf0b-bdb8-11ed-9185-b6be2f39279c
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'cbb_game_pbp',
    'function': 'getPlaybyPlay',
    'arguments': {
      'cbb_game_id': 'd85cdf0b-bdb8-11ed-9185-b6be2f39279c',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
