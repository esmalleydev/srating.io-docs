The cbb_game table will contain game information.

`home_team_rating` and `away_team_rating` is the estimated win percentage for the game.

  "cbb_game_id": "b58918bc-d155-11ed-9185-b6be2f39279c",
  "season": "2023",
  "away_team_id": "885557ec-8549-11ed-bf01-5296e1552828",
  "home_team_id": "75a9cb12-8549-11ed-bf01-5296e1552828",
  "network": "CBS",
  "home_team_rating": null,
  "away_team_rating": null,
  "away_score": 66,
  "home_score": 78,
  "status": "final",
  "current_period": "Final",

  "start_datetime": "2023-04-04T01:20:00.000Z",
  "start_timestamp": 1680571200,
  "updated_at": "2023-04-04T03:38:01.000Z",
  "attendance": 72423,
  "neutral_site": 1,

Currently game data is available beginning from the 2011 season to now.
