## Get Odds

This will return the current odds for a specified `game_id`.

[See the odds](/odds) sections to grab pre game or more historical odds.

### Required arguments
`game_id`

### Request

GET

```console
https://api.srating.io/v1/game/getOdds?game_id=40fd1ac3-5dd9-11ef-9222-9667e47de71a
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'game',
    'function': 'getOdds',
    'arguments': {
      'game_id': '40fd1ac3-5dd9-11ef-9222-9667e47de71a',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
