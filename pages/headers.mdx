## Request / Response Headers

### Required Request Headers
```console
X-API-KEY
```
You must include your API key in the header under "X-API-KEY"

### Important Response Headers
```console
X-REQUESTS-REMAINING
X-REQUESTS-USED
```
In our response we will include your remaining requests and a count of used requests. This data can also be viewed in your account, in the subscription.

Requests reset on the 1st of each month.
