The cbb_ranking table will contain ranking data for every team in the season. Rankings are usually calculated once per day in the morning.

You can use `current` to find the most recent calculated row in a season.

### Example row
```json
{
  "cbb_ranking_id": "faa69fe1-d29e-11ed-9185-b6be2f39279c",
  "season": 2023,
  "team_id": "87019264-8549-11ed-bf01-5296e1552828",
  "composite_rank": 6,
  "elo_rank": 17,
  "kenpom_rank": 7,
  "srs_rank": 11,
  "ap_rank": 3,
  "coaches_rank": 3,
  "net_rank": 5,
  "date_of_rank": "2023-04-04T04:00:00.000Z",
  "current": 1,
  "guid": null,
  "deleted": 0
}
```

### Schema
```console
+----------------+--------------+
| Field          | Type         |
+----------------+--------------+
| cbb_ranking_id | char(36)     |
| season         | int unsigned |
| team_id        | char(36)     |
| composite_rank | int unsigned |
| elo_rank       | int unsigned |
| kenpom_rank    | int unsigned |
| srs_rank       | int unsigned |
| ap_rank        | int unsigned |
| coaches_rank   | int unsigned |
| net_rank       | int unsigned |
| date_of_rank   | date         |
| current        | tinyint(1)   |
| guid           | char(36)     |
| deleted        | tinyint(1)   |
+----------------+--------------+
```
