## Get a player's game log

This request returns the player boxscores for a season along with the game information.

### Required arguments
`player_id` and `season`

### Request

GET

```console
https://api.srating.io/v1/player/getGameLogs?player_id=dff6dce2-ad5a-11ed-9185-b6be2f39279c&season=2023
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'player',
    'function': 'getGameLogs',
    'arguments': {
      'player_id': 'dff6dce2-ad5a-11ed-9185-b6be2f39279c',
      'season': 2023,
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
