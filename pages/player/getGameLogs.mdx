## Get a player's game log

This request returns the player boxscores for a season along with the game information.

### Usage
This will count as 15 usage tokens.

The same data can be retrieved with other relevant get requests, but typically this request will be more efficent.

### Required arguments
`organization_id`, `division_id`, `player_id`, `season`

### Request

GET

```console
https://api.srating.io/v1/player/getGameLogs?organization_id=f1c37c98-3b4c-11ef-94bc-2a93761010b8&division_id=bf602dc4-3b4a-11ef-94bc-2a93761010b8&player_id=dff6dce2-ad5a-11ed-9185-b6be2f39279c&season=2023
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'player',
    'function': 'getGameLogs',
    'arguments': {
      'organization_id': 'f1c37c98-3b4c-11ef-94bc-2a93761010b8',
      'division_id': 'bf602dc4-3b4a-11ef-94bc-2a93761010b8',
      'player_id': 'dff6dce2-ad5a-11ed-9185-b6be2f39279c',
      'season': 2023,
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
