## Get an ELO row

This request will find the first matching row and typically only used when passing in the `elo_id` or `organization_id`, `team_id` and `season` argument.

### Request

GET

```console
https://api.srating.io/v1/elo/get?organization_id=f1c37c98-3b4c-11ef-94bc-2a93761010b8&team_id=87019264-8549-11ed-bf01-5296e1552828&season=2023&current=1
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'elo',
    'function': 'get',
    'arguments': {
      'organization_id': 'f1c37c98-3b4c-11ef-94bc-2a93761010b8',
      'team_id': '87019264-8549-11ed-bf01-5296e1552828',
      'season': 2023,
      'current': 1,
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
