## Get a player boxscore row

This request will find the first matching row and typically only used when passing in the `player_boxscore_id` or `organization_id`, `game_id` and `player_id` argument.

### Request

GET

```console
https://api.srating.io/v1/player_boxscore/get?organization_id=f1c37c98-3b4c-11ef-94bc-2a93761010b8&game_id=0f0fb2e7-5dd9-11ef-9222-9667e47de71a&player_id=e27545f0-ad5a-11ed-9185-b6be2f39279c
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'player_boxscore',
    'function': 'getPlayerBoxscore',
    'arguments': {
      'organization_id': 'f1c37c98-3b4c-11ef-94bc-2a93761010b8',
      'game_id': '0f0fb2e7-5dd9-11ef-9222-9667e47de71a',
      'player_id': 'e27545f0-ad5a-11ed-9185-b6be2f39279c',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
