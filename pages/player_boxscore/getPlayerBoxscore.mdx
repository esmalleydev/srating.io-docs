## Get a player boxscore row

This request will find the first matching row and typically only used when passing in the `cbb_player_boxscore_id` or `cbb_game_id` and `player_id` argument.

### Request

GET

```console
https://api.srating.io/v1/cbb_player_boxscore/get?cbb_game_id=b58918bc-d155-11ed-9185-b6be2f39279c&player_id=4ce4ef10-ad5a-11ed-9185-b6be2f39279c
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'cbb_player_boxscore',
    'function': 'get',
    'arguments': {
      'cbb_game_id': 'b58918bc-d155-11ed-9185-b6be2f39279c',
      'player_id': '4ce4ef10-ad5a-11ed-9185-b6be2f39279c',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
