The coach table will contain coach information

### Example row
```json
{
  "coach_id": "cf3f6a9d-0fe0-11ef-9686-72fab666226a",
  "first_name": "Matt",
  "last_name": "Painter",
  "cbb": 1,
  "cfb": 0,
  "nba": 0,
  "nfl": 0,
  "nhl": 0,
  "guid": "0e8f0ea6-0042-43b5-85c0-72ea08875219",
  "deleted": 0
}
```

```console
+-------------------------+--------------+
```
