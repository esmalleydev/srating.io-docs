All sports share the same tables, for most requests, you will need to specify a `organization_id` and `division_id`.

The `organization_id` is the sport, like NCAA mens basketball or football.

The `division_id` is the division, like D1 or FBS.

You can send a `read` request to get all the active rows in each table

### Organization request
GET

```console
https://api.srating.io/v1/organization/read?inactive=0
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'organization',
    'function': 'read',
    'arguments': {
      'inactive': '0',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```

```console
https://api.srating.io/v1/division/read?inactive=0
```
