The `odds` table will contain the money line, spread, over / under information for a game.

A row will not be updated, if the odds change, a new row will be created.

You can find the current odds by specifing `current` = 1 and use `live` to determine if the odds are before or during the game.

### Example row
```json
{
  "odds_id": "209de407-5f70-11ef-9222-9667e47de71a",
  "game_id": "40fd1ac3-5dd9-11ef-9222-9667e47de71a",
  "money_line_away": 828,
  "money_line_home": -3453,
  "spread_away": 13,
  "spread_home": -11,
  "over": 129.5,
  "under": 145.5,
  "date_of_entry": "2023-04-03T08:00:00.000Z",
  "live": 1,
  "current": 1,
  "guid": "d9db673c-ce69-4581-a1ec-a36ec273c6eb",
  "deleted": 0,
}
```

Historical odds may be missing or incomplete.
