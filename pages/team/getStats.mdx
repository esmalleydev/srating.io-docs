## Team statistics
Returns statistics for a team in a specified season.

### Required arguments
`team_id` and `season`

### Request

GET

```console
https://api.srating.io/v1/team/getStats?team_id=87019264-8549-11ed-bf01-5296e1552828&season=2023
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'team',
    'function': 'getStats',
    'arguments': {
      'team_id': '87019264-8549-11ed-bf01-5296e1552828',
      'season': 2023,
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
