## Team statistics
Returns statistics for a team in a specified season. This is also a great way to look at historical data.

If `current` = 1, this is the most recent ranking row for that team in the season.

You can use the `date_of_rank` to order the rankings, there should be 1 row per day during the season.

The stats are stored as averages based on the `games`. Each stat should have a corresponding rank out of all D1 teams, which also have a `date_of_rank` on the same date.

### Required arguments
`team_id` and `season`

### Request

GET

```console
https://api.srating.io/v1/team/getStats?team_id=87019264-8549-11ed-bf01-5296e1552828&season=2023
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'team',
    'function': 'getStats',
    'arguments': {
      'team_id': '87019264-8549-11ed-bf01-5296e1552828',
      'season': 2023,
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
