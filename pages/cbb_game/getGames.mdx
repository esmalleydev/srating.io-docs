## Get games
The getGames call will return the games along with [odds](/cbb_game_odds/get), [teams](/team/get), [ranking](/cbb_ranking/get) data.

### Usage
This will count as 40 usage tokens.

The same data can be retrieved with other relevant get requests, but typically this request will be more efficent when there are more than 4 games in a day.

### Required arguments
Must specify a start_date or a season with a home/away_team_id

`start_date` (YYYY-MM-DD)

OR

`season` AND (`home_team_id` OR `away_team_id`)

### Request

GET

```console
https://api.srating.io/v1/cbb_game/getGames?start_date=2023-04-01
```

POST
```js {5} copy filename="javascript"
fetch('https://api.srating.io/v1', {
  'method': 'POST',
  'headers': {
    'Content-Type': 'application/json',
    'X-API-KEY': 'YOUR_API_KEY',
  },
  'body': JSON.stringify({
    'class': 'cbb_game',
    'function': 'getGames',
    'arguments': {
      'start_date': '2023-04-01',
    }
  }),
}).then(response => {
  return response.json();
}).then(json => {
  console.log(json);
  return json;
}).catch(error => {
  console.log(error);
});
```
